Реализовать полнофункциональную систему тестирования.

В системе должны быть 2 режима: Администратор и Студент.

Тестирование может осуществляться по различным категориям знаний.
Например: [раздел] Математика: [тест] Дискретная математика; [тест] Математический Анализ; ...
          [раздел] Физика    : [тест] Квантовая физика     ; [тест] Механика             ; ...

Описание режима работа для Студента:
:: Для входа в систему студент должен зарегистрироваться, в дальнейшем вход по логину и паролю.
! Логины всех пользователей должны быть уникальными, хранятся логины и пароли в зашифрованном виде.
:: При регистрации нужно указать: Логин*, Пароль*, Ф.И.О., Почтовый адрес, Телефон.
! *Обязательные поля которые студент не сможет редактировать самостоятельно.
:: После входа студент имеет возможность просмотреть свои предыдущие результаты тестирования, сдать новое тестирование.
:: После сдачи теста студент видит результат тестирования, количество/процент правильных ответов и полученную оценку.
! Оценивание нужно вести на основании 12 балльной системы, привязанной к количеству вопросов теста.
? Тест имеет один из двох статусов: paused - приостановлен; completed - завершен.
:: Студент имеет возможность прервать тестирование и продолжить его тогда, когда ему это будет удобно.

Описание режима работы для Администратора:
:: В системе может быть только один админ, логин и пароль админа задаётся при первом входе в программу.
:: Управление пользователями — создание, удаление, модификация.
:: Просмотр статистики — просмотр результатов тестирования по разделам, по тестам, по пользователям. Статистику можно сохранить в txt файл.
:: Управление тестированием — админ может добавлять новые или редактировать существующие: разделы, тесты, вопросы и ответы к тестам.
! Имеется возможность импорта из файла и экспорт в файл отдельных разделов или готовых тестов.

Общие критерии:
! Необходимо в полной мере использовать ООП и STL.
! Код должен быть читабельным, хорошо прокомментированным и использовать единую систему именования идентификаторов.
